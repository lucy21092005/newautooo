## 🔧 Key Implementation Details

### Guard Against Duplicate Processes
```python
def play_alarm(self):
    # If process already running, skip
    if self.alarm_process is not None and self.alarm_process.poll() is None:
        return  # ← Only ONE subprocess at a time
```

### Graceful + Force Kill Strategy
```python
def stop_alarm(self):
    try:
        self.alarm_process.terminate()      # Graceful first
        self.alarm_process.wait(timeout=1)
    except subprocess.TimeoutExpired:
        self.alarm_process.kill()           # Force if needed
        self.alarm_process.wait(timeout=1)
    self.alarm_process = None               # Always reset
```

### One-Time Trigger with Hysteresis
```python
if risk >= 70:
    if not self.alarm_on:                  # ← Guard: only once!
        self.alarm_on = True
        self.play_alarm()                  # ← Called ONCE per entry
else:
    if self.alarm_on:
        self.alarm_on = False
        self.stop_alarm()
    
    if risk < 60:                          # ← Hysteresis: re-arm at 60
        self.alarm_triggered_this_session = False
```
